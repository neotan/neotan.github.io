express-react-generator-typescript

Express application generator, with TypeScript, Jest, ESLint and optionally React support, which’s inspired by Sean Maxwell’s express-generator-typescript.

Installation

1

$ npm install -g express-react-generator-typescript

Quick Start

Create the app:

1
2
3
4
5
6

$ ergt my-app

// Or add TypeScript support

$ ergt -t my-app

Start your app at http://localhost:5555/:

1

$ cd my-app && npm i && npm run dev

Command Line Options

This generator can also be further configured with the following command line flags.

    -V  --version         output the current version
    -t  --typescript      add TypeScript support
    -f  --force           force on non-empty directory

Why these packages

  1. express
  2. ts-node: Running .ts files directly

  4. eslint 1. TSLint will be replaced by ESLint` 1. Formatting codes with prettier together

  6. reflect-metadata: Supporting Decorator
  7. jest
  8. ts-jest: Running .test.ts files directly
  9. supertest: Testing API endpoints
  10. nodemon Watching files changes then live reloading
  11. husky Automatically test and lint before Git commit/push
