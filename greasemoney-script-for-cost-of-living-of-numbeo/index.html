Cost of living data comparison script

Description

A GreaseMonkey script for visualizing multiple cities' data from www.numbeo.com/cost-of-living/

Usage

  1. Install Tampermonkey
  2. Install this script
  3. Open www.numbeo.com or its sub routes
  4. Will see Charts and Table be attached on the page
  5. Change the city names(split them with comma) in Cities field
  6. Press Enter key or click the Submit button
